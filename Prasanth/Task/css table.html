CSS Grid - Properties

Hi, This a webpage created to showcase the CSS Grid properties. This page has been created using a CSS Grid.

The left side section is created using CSS Flexbox Property. The flex direction is set as Column.

The Box Shadow property has been used in the Header section. The shadow is highlighted in green color.

The Border Radius Property is used in this main section and the right side ad section, you can see the border highlighted in red and blue color with border radius set as 50px, 20px and 25px respectively.
